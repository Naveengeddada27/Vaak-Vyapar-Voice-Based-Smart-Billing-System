# Vaak Vyapar – Voice-Based Smart Billing System

Vaak Vyapar is an AI-powered voice-based smart billing system designed to simplify retail billing through speech commands. Users can create and manage bills using voice input, enabling faster and more efficient transactions.

## Features
- 🎙️ Voice-controlled billing
- 🛒 Add and remove items using speech commands
- 📄 Automatic invoice generation
- 📊 Real-time bill calculation
- 🗂️ Transaction history management
- 🔒 User-friendly and efficient interface

## Tech Stack
- React.js
- JavaScript
- HTML
- CSS
- SQLite

## Applications
- Retail stores
- Supermarkets
- Grocery shops
- Small businesses

## Future Enhancements
